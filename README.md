# Crypto-API

---
## Project Overview
Crypto-API is a data workflow project that fetches cryptocurrency market data via API calls, preprocesses the data using Python, and visualizes trends with Power BI. The project demonstrates an end-to-end pipeline from data acquisition to analysis-ready visuals.

---
## Key Features
- Fetches live cryptocurrency data from public APIs
- Cleans and preprocesses raw JSON or CSV data using Python
- Prepares data tables for Power BI visualization
- Creates interactive dashboards for market analysis

---
## 🛠️ Technologies Used

- **Python**  
  - API Data Extraction
  - Data Cleaning
  - Data Preprocessing
  - Data Transformation

- **Libraries**
  - Pandas
  - NumPy
  - Requests
  - Matplotlib / Seaborn (if used)

- **Power BI**
  - Interactive Dashboard Creation
  - Data Visualization
  - Trend Analysis

- **API**
  - Cryptocurrency market data API


---
## Workflow
1. Fetch crypto data from an API endpoint
2. Parse and clean the returned data
3. Transform the dataset into analysis-ready tables
4. Export or save processed data for Power BI
5. Build Power BI visuals using the cleaned dataset

---

## 🔄 Project Workflow

### 1. Data Collection (API Integration)

- Fetched real-time cryptocurrency data using API requests.
- Extracted important attributes such as:
  - Cryptocurrency Name
  - Current Price
  - Market Capitalization
  - Trading Volume
  - Price Changes
  - Market Rank

The API response was collected in JSON format and converted into a structured dataset.

---

### 2. Data Preprocessing Using Python

The raw API data was processed using Python.

Performed operations such as:

- Handling missing values
- Removing duplicate records
- Data type conversion
- Feature selection
- Data formatting
- Creating calculated fields
- Preparing data for visualization
